# ML-Based Return Prediction for Green Energy Stocks

## Overview

This project develops a machine learning framework for predicting next-day returns of green-energy stocks using historical market data and technical indicators. Multiple regression models are implemented and compared to evaluate their predictive performance on financial time-series data.

The project applies feature engineering, model evaluation, and explainable AI techniques to understand the key factors influencing stock returns.

---

## Objectives

- Predict next-day stock returns using machine learning.
- Compare multiple regression models.
- Engineer technical indicators from historical market data.
- Evaluate model performance using statistical metrics.
- Interpret model predictions using feature importance and SHAP analysis.

## Results

The ensemble models outperformed traditional regression techniques, with XGBoost and Random Forest achieving the strongest predictive performance. Feature importance analysis identified lagged returns, volatility, RSI, and momentum as the most influential variables.
